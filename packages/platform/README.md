# n8n-nodes-agent-brains

AgentBrains action nodes for n8n. This package bundles the Knowledge Base, Employee, RAG, and Synthetic QA nodes into a single installable npm package.

## Install

In n8n, open **Settings -> Community Nodes -> Install** and paste:

    n8n-nodes-agent-brains

Or from the CLI:

    npm install n8n-nodes-agent-brains

## Included Nodes

- **AgentBrains Knowledge Base**
- **AgentBrains Employee**
- **AgentBrains RAG**
- **AgentBrains Synthetic QA**

## Credentials

Requires an AgentBrains API key. Generate one in the AgentBrains Control Panel, then paste it into the **AgentBrains Integration API** credential in n8n.

Step-by-step: <https://docs.agent-brains.com/integrations/access-api-key>
